

\section{Introduction}

Let $\Gamma$ be a finite group with a subset S. The \emph{Cayley digraph}, denoted Cay($\Gamma$,S), is a digraph with vertex set $\Gamma$, and (x,y) is a directed edge if and only if $yx^{-1}$ $\in$ S.
